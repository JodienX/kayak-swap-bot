## Auto-swap-Tx

- Open [Codespace](https://github.com/codespaces) or Gitpod
- Use this command to run
```bash
wget https://raw.githubusercontent.com/JodienX/morph-auto-tx/main/script.sh
```
```bash
chmod +x script.sh
```
```bash
./script.sh
```
- Done
